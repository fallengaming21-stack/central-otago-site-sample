# AI Image Enrichment Plan (No Upfront Cost)

## Goal
Create premium-looking visual assets quickly while waiting for real business photography.

## Strategy
1. Generate **concept images** for hero/gallery using free AI endpoints (Pollinations).
2. Use them as temporary visual polish with clear replacement plan.
3. Replace with authentic venue photos as soon as client provides them.

## Prompt set (Tin Goose Cafe)
- Modern scandi cafe interior, warm morning light, oak textures, espresso machine, high-end editorial photo
- Flat white and pastry on ceramic plate, soft natural light, premium food photography style
- Cozy brunch table in New Zealand cafe, neutral palette, candid lifestyle composition
- Cafe counter cabinet with pastries and scones, tasteful styling, depth of field
- Street-facing cafe entrance in small NZ town, clean signage, golden hour look
- Friendly barista serving coffee in modern local cafe, documentary style

## Usage guidance
- Keep color grading consistent with site palette (#F7F2EA / #2F211A / #B5793E)
- Avoid uncanny faces in hero; prioritize interiors/food/product shots
- Compress assets to webp/jpg < 300KB each

## Risk note
AI visuals can misrepresent venue details. Use as concept visuals only until replaced with real photos.
